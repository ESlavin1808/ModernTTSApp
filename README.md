# 🎙️ Voice Studio

**Современное десктопное приложение для синтеза (TTS) и распознавания речи (STT) на Windows.**

## ✨ Возможности

### 🎤 TTS (Синтез речи)

| Функция | Описание |
|---------|----------|
| **Движок** | Microsoft Edge TTS (онлайн) |
| **Голоса** | 8+ голосов: английские (Aria, Guy, Jenny), русские (Дмитрий, Светлана), немецкий, французский, испанский |
| **Скорость** | Регулировка от −50% до +50% |
| **Плеер** | Встроенное воспроизведение (Play/Stop) |
| **Сохранение** | Экспорт в MP3 или WAV |
| **Буфер обмена** | Вставка текста одной кнопкой 📋 |

### 🎧 STT (Распознавание речи)

| Функция | Описание |
|---------|----------|
| **Движок** | OpenAI Whisper (локально, без API-ключей) |
| **Модели** | tiny / base / small / medium / large |
| **Запись** | Встроенный рекордер с индикатором уровня звука и таймером |
| **Файлы** | Загрузка аудио через диалог или **Drag & Drop** |
| **Результат** | Копирование в буфер или сохранение в TXT |

### 📜 История

- Все синтезы и транскрибации сохраняются автоматически
- **Пагинация** — по 20 записей на страницу (поддерживается 500+)
- Воспроизведение аудио из истории
- Удаление отдельных записей или очистка всего списка

### 🌐 Локализация

- **Русский** и **Английский** интерфейсы
- Переключение языков одной кнопкой
- Язык сохраняется между сессиями

### 🔒 Безопасность

- **Санитизация SSML** — предотвращает XML-инъекции через ввод текста
- **Валидация JSON** — повреждённые файлы истории не крашат приложение
- **Проверка путей** — безопасное сохранение файлов

### ⌨️ Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `Ctrl+Enter` | Синтезировать речь |
| `Ctrl+R` | Старт/стоп записи |
| `Ctrl+1` | Переключиться на TTS |
| `Ctrl+2` | Переключиться на STT |
| `Ctrl+3` | Переключиться на историю |
| ПКМ | Контекстное меню (вырезать/копировать/вставить) |

### ⚡ Оптимизация

- **Кеширование Whisper** — модель загружается один раз при старте
- **Асинхронное сохранение** — история не блокирует интерфейс
- **Debounce слайдера** — 50 мс задержка для плавности
- **Чистый код** — удалён мёртвый код, лимит истории 500 записей

## 🚀 Установка

### Требования

- **Python 3.8+** (рекомендуется 64-битный)
- **Windows 10/11**
- Подключение к интернету (для Edge TTS и загрузки моделей Whisper)

### Быстрый старт

```bash
# 1. Клонировать репозиторий
git clone https://github.com/ВАШ_АККАУНТ/voice-studio.git
cd voice-studio

# 2. Создать виртуальное окружение
python -m venv venv

# 3. Активировать
venv\Scripts\activate

# 4. Установить зависимости
pip install -r requirements.txt

# 5. Запустить
python main.py
```

### Запуск без консоли

Просто дважды кликните **`run.bat`** — он активирует окружение и запустит приложение без командной строки.

## ⚙️ Настройки

Настройки сохраняются автоматически в `settings.json`:

- **Положение окна** — размер и позиция восстанавливаются при следующем запуске
- **Язык** — русский или английский
- **Голос** — последний выбранный голос TTS
- **Модель Whisper** — последняя выбранная модель
